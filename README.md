# Vortex Tech App Dev Internship — Week 2

## To-Do List App (Flutter)

A working to-do list app built with Flutter. Users can add tasks, mark them complete, and delete them, using `setState` for state management.

## Features

- Add a task via text field + button
- Mark tasks complete with a checkbox (strikethrough style)
- Delete tasks via swipe or delete icon
